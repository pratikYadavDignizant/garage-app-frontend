# Technical Stack

This document outlines the technologies and tools used in the Garage Management Web App.

## Core Framework & Language

- **[Next.js 16.1.1 (App Router)](https://nextjs.org/)**: The React framework for production, providing server-side rendering, static site generation, and optimized performance.
- **[TypeScript](https://www.typescriptlang.org/)**: A typed superset of JavaScript that ensures code quality and developer productivity.

## Frontend Technologies

- **[Tailwind CSS 4](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
- **[Radix UI](https://www.radix-ui.com/)**: Low-level, accessible UI primitive components (Dialog, Label, Slot, etc.).
- **[Lucide React](https://lucide.dev/)**: High-quality, consistent icons for the application.
- **[TanStack Query (React Query) v5](https://tanstack.com/query/latest)**: Powerful asynchronous state management for fetching, caching, and updating data.
- **[React Hook Form](https://react-hook-form.com/)**: Performant, flexible, and extensible forms with easy-to-use validation.
- **[Zod](https://zod.dev/)**: TypeScript-first schema declaration and validation library.

## Authentication & Communication

- **[Firebase Authentication](https://firebase.google.com/docs/auth)**: Handles secure user authentication via Phone OTP.
- **[Axios](https://axios-http.com/)**: Promised-based HTTP client for making API requests to the backend.

## Visualization & Utilities

- **[Recharts](https://recharts.org/)**: A composable charting library built on React components.
- **[date-fns](https://date-fns.org/)**: Modern JavaScript date utility library.
- **[Sonner](https://sonner.emilkowal.ski/)**: An opinionated toast component for React.
- **[CLS-X & Tailwind Merge](https://github.com/lukeed/clsx)**: Utilities for efficiently merging Tailwind CSS classes.

## Development Tools

- **ESLint**: Pluggable linting utility for JavaScript and TypeScript.
- **PostCSS**: A tool for transforming CSS with JavaScript plugins.
